# Pcard

## Development

### Install Dependencies

php backend via composer:

```cmd
cd server/public
composer install
```

angular frontend via npm:

```cmd
cd webapp
npm install
```

### Run

To run php backend use:

```cmd
cd server/public
php -S localhost:8888 .\index_for_built_in_php.php
```

To run angular frontend use:

```cmd
cd webapp
ng serve
```
